# 🩸 Blood Donation Portal

A simple web-based Blood Donation Portal that connects blood donors with people in need of blood. This project helps in managing donor details and blood requests efficiently.

---

## 🚀 Features

- Donor registration system  
- Blood request submission  
- Search donors by blood group & location  
- Admin dashboard (optional)  
- Contact / request management  
- Mobile-friendly UI  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / PHP  
- Database: MySQL / Firebase  

---

## ⚙️ Installation & Setup

git clone http://localhost/blood_donation

cd blood_donation_portal

npm install

npm start

Open in browser:
http://localhost:3000
